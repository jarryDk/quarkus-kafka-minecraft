= Quarkus-Kafka

== Build application

.Dev mode
[source,bash]
----
mvn quarkus:dev
----

.JVM mode
[source,bash]
----
mvn clean package
----

.Native mode
[source,bash]
----
mvn clean package -Pnative
----

== Start application

.JVM mode
[source,bash]
----
java -jar target/quarkus-app/quarkus-run.jar
----

.Native mode
[source,bash]
----
target/quarkus-kafka-1.0.0-SNAPSHOT-runner
----


== Get players activity via curl

[source,bash]
----
curl -N http://localhost:8080/players
----
